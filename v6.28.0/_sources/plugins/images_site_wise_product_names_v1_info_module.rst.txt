cisco.dnac.images_site_wise_product_names_v1_info module -- Information module for Images Site Wise Product Names V1
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

    This module is part of the `cisco.dnac collection <https://galaxy.ansible.com/cisco/dnac>`_ (version 6.28.0).

Synopsis
--------

.. Description

- Get all Images Site Wise Product Names V1.
- Returns a list of network device product names and associated sites for a given image identifier. Refer \`/dna/intent/api/v1/images\` API for obtaining \`imageId\`.


.. note::
    This module has a corresponding :ref:`action plugin <action_plugins>`.

Requirements
------------
The below requirements are needed on the host that executes this module.

- dnacentersdk \>= 2.4.9
- python \>= 3.5

.. Notes

Notes
-----

.. note::
   - SDK Method used are software\_image\_management\_swim.SoftwareImageManagementSwim.retrieves\_network\_device\_product\_names\_assigned\_to\_a\_software\_image\_v1,
   - Paths used are get /dna/intent/api/v1/images/{imageId}/siteWiseProductNames,
   - Supports \ :literal:`check\_mode`\ 
   - The plugin runs on the control node and does not use any ansible connection plugins, but instead the embedded connection manager from Cisco DNAC SDK
   - The parameters starting with dnac\_ are used by the Cisco DNAC Python SDK to establish the connection

See Also
--------
