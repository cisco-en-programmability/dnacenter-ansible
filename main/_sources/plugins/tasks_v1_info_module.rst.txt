.. Title

cisco.dnac.tasks_v1_info module -- Information module for Tasks V1
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This module is part of the `cisco.dnac collection <https://galaxy.ansible.com/cisco/dnac>`_ (version 6.28.0).

    To install it, use: :code:`ansible-galaxy collection install cisco.dnac`.
    You need further requirements to be able to use this module,
    see :ref:`Requirements <ansible_collections.cisco.dnac.tasks_v1_info_module_requirements>` for details.

    To use it in a playbook, specify: :code:`cisco.dnac.tasks_v1_info`.

.. version_added

.. rst-class:: ansible-version-added

New in cisco.dnac 6.15.0

.. contents::
   :local:
   :depth: 1

.. Deprecated


Synopsis
--------

.. Description

- Get all Tasks V1.
- Get Tasks V1 by id.
- Returns tasks based on filter criteria.
- Returns the task with the given ID.

.. note::
    This module has a corresponding :ref:`action plugin <action_plugins>`.

Requirements
------------
The below requirements are needed on the host that executes this module.

- dnacentersdk \>= 2.4.9
- python \>= 3.5

.. Notes

Notes
-----

.. note::
   - SDK Method used are task.Task.get\_tasks\_by\_id\_v1, task.Task.get\_tasks\_v1,
   - Paths used are get /dna/intent/api/v1/tasks, get /dna/intent/api/v1/tasks/{id},
   - Supports \ :literal:`check\_mode`\ 
   - The plugin runs on the control node and does not use any ansible connection plugins, but instead the embedded connection manager from Cisco DNAC SDK
   - The parameters starting with dnac\_ are used by the Cisco DNAC Python SDK to establish the connection

Examples
--------

.. code-block:: yaml+jinja

    
    - name: Get all Tasks V1
      cisco.dnac.tasks_v1_info:
        dnac_host: "{{dnac_host}}"
        dnac_username: "{{dnac_username}}"
        dnac_password: "{{dnac_password}}"
        dnac_verify: "{{dnac_verify}}"
        dnac_port: "{{dnac_port}}"
        dnac_version: "{{dnac_version}}"
        dnac_debug: "{{dnac_debug}}"
        headers: "{{my_headers | from_json}}"
        offset: 0
        limit: 0
        sortBy: string
        order: string
        startTime: 0
        endTime: 0
        parentId: string
        rootId: string
        status: string
      register: result

    - name: Get Tasks V1 by id
      cisco.dnac.tasks_v1_info:
        dnac_host: "{{dnac_host}}"
        dnac_username: "{{dnac_username}}"
        dnac_password: "{{dnac_password}}"
        dnac_verify: "{{dnac_verify}}"
        dnac_port: "{{dnac_port}}"
        dnac_version: "{{dnac_version}}"
