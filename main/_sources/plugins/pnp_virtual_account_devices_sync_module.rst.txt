.. Title

cisco.dnac.pnp_virtual_account_devices_sync module -- Resource module for Pnp Virtual Account Devices Sync
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This module is part of the `cisco.dnac collection <https://galaxy.ansible.com/cisco/dnac>`_ (version 6.18.0).

    To install it, use: :code:`ansible-galaxy collection install cisco.dnac`.
    You need further requirements to be able to use this module,
    see :ref:`Requirements <ansible_collections.cisco.dnac.pnp_virtual_account_devices_sync_module_requirements>` for details.

    To use it in a playbook, specify: :code:`cisco.dnac.pnp_virtual_account_devices_sync`.

.. version_added

.. rst-class:: ansible-version-added

New in cisco.dnac 3.1.0

.. contents::
   :local:
   :depth: 1

.. Deprecated


Synopsis
--------

.. Description

- Manage operation create of the resource Pnp Virtual Account Devices Sync.
- Synchronizes the device info from the given smart account & virtual account with the PnP database. The response payload returns a list of synced devices Deprecated .


.. note::
    This module has a corresponding :ref:`action plugin <action_plugins>`.

Requirements
------------
The below requirements are needed on the host that executes this module.

- dnacentersdk \>= 2.7.2
- python \>= 3.5

.. Notes

Notes
-----

.. note::
   - SDK Method used are device\_onboarding\_pnp.DeviceOnboardingPnp.sync\_virtual\_account\_devices,
   - Paths used are post /dna/intent/api/v1/onboarding/pnp-device/vacct-sync,
   - Does not support \ :literal:`check\_mode`\ 
   - The plugin runs on the control node and does not use any ansible connection plugins, but instead the embedded connection manager from Cisco DNAC SDK
   - The parameters starting with dnac\_ are used by the Cisco DNAC Python SDK to establish the connection

See Also
--------

   `Cisco DNA Center documentation for Device Onboarding (PnP) SyncVirtualAccountDevices <https://developer.cisco.com/docs/dna-center/#!sync-virtual-account-devices>`_
       Complete reference of the SyncVirtualAccountDevices API.
