cisco.dnac.pnp_device_count_v1_info module -- Information module for Pnp Device Count V1

Note

This module is part of the cisco.dnac collection (version 6.29.0).

To install it, use: ansible-galaxy collection install cisco.dnac. You need further requirements to be able to use this module, see Requirements for details.

To use it in a playbook, specify: cisco.dnac.pnp_device_count_v1_info.

Synopsis

  • Get all Pnp Device Count V1.

  • Returns the device count based on filter criteria. This is useful for pagination.

Note

This module has a corresponding action plugin.

Requirements

The below requirements are needed on the host that executes this module.

  • dnacentersdk >= 2.4.9

  • python >= 3.5
